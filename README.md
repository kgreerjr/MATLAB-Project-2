# MATLAB-Project-2

A MATLAB project applying Singular Value Decomposition (SVD) to perform image compression, evaluating matrix approximations at varying rank levels and measuring accuracy through RMSE.

## About

- Applied SVD to decompose image matrices into rank-k approximations at multiple compression levels, reconstructing images from reduced singular value representations.
- Computed Root Mean Square Error (RMSE) at each rank to quantify the trade-off between compression ratio and image fidelity, identifying the point of diminishing returns.
- Demonstrated a practical application of linear algebra to data compression, using MATLAB to visualize the relationship between matrix rank and approximation quality.
